# 智能学术助手

一个具备论文润色与文献阅读辅助功能的智能系统。

## 功能特点

- **论文润色**：支持表达润色、逻辑检查、去AI味、中英文翻译、缩写和扩写等功能，**新增中文语言支持**
- **文献分析**：支持PDF和DOCX文件上传，提取关键信息，生成大纲，提供内容解释
- **API设置**：支持配置自定义API密钥，接入不同的大模型

## 快速开始

### 系统要求

- Python 3.7+
- Node.js 14+
- npm 6+

### 安装步骤

1. **克隆项目**（如果尚未克隆）

2. **安装后端依赖**
   ```bash
   cd 项目根目录
   pip install -r requirements.txt
   ```

3. **安装前端依赖**
   ```bash
   cd frontend
   npm install
   ```

4. **配置API密钥**
   - 在`.env`文件中设置API密钥，或在前端界面的"API设置"中配置

### 启动系统

最简单的方法是使用提供的启动脚本：

1. 双击运行 `start_app.bat` 文件
2. 等待服务启动完成
3. 在浏览器中访问前端服务地址（默认：http://localhost:3000）

### 手动启动（可选）

**启动后端服务**：
```bash
cd d:\paper agent\src4
python -m uvicorn app.main:app --reload
```

**启动前端服务**：
```bash
cd d:\paper agent\src4\frontend
npm run dev
```

## 使用指南

### 论文润色
1. 在左侧导航栏选择"论文润色"
2. 选择润色类型
3. 选择语言（支持中文和英文）
4. 输入要润色的文本
5. 点击"润色"按钮
6. 查看润色结果和修改日志

### 文献分析
1. 在左侧导航栏选择"文献分析"
2. 上传PDF或DOCX文件
3. 点击"分析"按钮
4. 查看分析结果，包括标题、作者、摘要、研究方法、研究结果、结论、关键词和大纲
5. 输入需要解释的内容，点击"解释内容"获取详细解释

### API设置
1. 在左侧导航栏选择"API设置"
2. 输入API密钥、Base URL和模型名称
3. 点击"保存设置"按钮

## 技术架构

- **后端**：FastAPI框架，Python
- **前端**：React框架，Vite构建工具
- **大模型**：支持DeepSeek等API
- **文档处理**：PyPDF2（PDF）、python-docx（DOCX）

## 常见问题

### 1. 启动失败
- 检查Python和Node.js是否正确安装
- 检查依赖是否已安装
- 检查端口是否被占用

### 2. API调用失败
- 检查API密钥是否正确
- 检查网络连接
- 检查Base URL是否正确

### 3. 文献分析结果不正确
- 确保上传的文件是有效的PDF或DOCX文件
- 确保文件内容清晰可识别
